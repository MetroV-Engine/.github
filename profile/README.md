## About MetroV

**MetroV Engine** is a 2D game engine specialized for building **Metroidvania-style games**.

Built to be easy to use without being limiting, MetroV is designed for both beginners and experienced developers. It provides intuitive tools that allow non-developers to create games without deep programming knowledge, while offering advanced control and flexibility for those who want to customize every aspect of gameplay, physics, and world design.

Whether you're an experienced engineer or a game designer with no coding background, MetroV makes creating immersive exploration games both accessible and powerful.
